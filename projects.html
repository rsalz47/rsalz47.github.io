Projects

I try my best to stay busy. Visit my github to see if I'm keeping myself honest.

As I mentioned on the landing page, most of my projects focus on some aspect of low-level programming and system design. For involved projects, I typically accompany the codebase with a blog post here. These can be iterative, evolving as I work through the project, or summarizing; I don't lean towards one style or the other. I try to treat these projects as practical learning experiences. As such, the writeups for these will be (perhaps pedantically) thorough, so that folks with minimal background in the subject can get a grasp of what's going on.

Like most anyone of this persuasion, my Github contains all of my work that I feel is worth sharing, so if you're looking for just the quick-and-dirty, then that's the place to be. I typically document my work fairly well, so you won't be missing out on much if you just browse the source instead of reading along a blog post.
